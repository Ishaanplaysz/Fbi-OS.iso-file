About FBI-OS

  • Multi-layer password protection (1-10 layers)
  • Green "FBI code" theme
  • File manager, app installer, TOR, Chrome, custom app downloader
  • Update button for auto-updating from GitHub
  • Command Prompt and more features coming soon

To update FBI-OS, use the built-in update button or run update_os.sh in your OS.

How to Use

  1. Download the ISO file above.
  2. Write it to a USB drive or use in a VM.
  3. Boot and follow on-screen instructions.
